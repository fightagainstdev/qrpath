# 膨胀小羊羔 

这是游戏 膨胀小羊羔 (by LUMIN)。

## 如何运行

直接在现代浏览器中打开 `index.html` 文件即可开始游戏！

## 游戏操作

- **移动**: 方向键 或 WASD
- **翻滚**: 空格键 或 CTRL
- **攻击**: 左键
- **蓄力攻击**: 按住左键
- **格挡**: 右键 或 SHIFT
- **暂停**: P 或 ESC
