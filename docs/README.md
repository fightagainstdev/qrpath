# 膨胀小羊羔 (Path to Glory - Chinese Version)

这是游戏 "Path to Glory" 的完整中文汉化版本，标题已改为 "膨胀小羊羔 (by LUMIN)"。

## 如何运行

直接在现代浏览器中打开 `index.html` 文件即可开始游戏！

## 游戏操作

- **移动**: 方向键 或 WASD
- **翻滚**: 空格键 或 CTRL
- **攻击**: 左键
- **蓄力攻击**: 按住左键
- **格挡**: 右键 或 SHIFT
- **暂停**: P 或 ESC

## 汉化内容

- 标题: PATH TO GLORY → 膨胀小羊羔 (by LUMIN)
- Logo: P ATH TO G LORY → 膨 胀 小 羊 羔
- 暂停界面: GAME P AUSED → 游 戏 暂 停
- Debug信息: Entities → 实体
- 所有教程文本和游戏内公告已翻译为中文
- 结局文本全部翻译为中文

## 技术信息

- 文件大小: index.html (17KB), game.zip (13KB)
- 这是使用js13k-compiler构建的优化版本
- 包含完整的游戏逻辑、图形和音频
